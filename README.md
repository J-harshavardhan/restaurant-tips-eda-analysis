# 📊 Restaurant Tips Analysis – Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs a **complete Exploratory Data Analysis (EDA)** on the **Restaurant Tips dataset** to uncover patterns that influence tipping behavior in restaurants.

The analysis follows a **structured 7-step EDA framework**, starting from understanding the problem to generating actionable insights.

The goal is to answer important business questions such as:

* What factors influence the tip amount?
* Do customers tip more during **lunch or dinner**?
* Does **party size** impact tipping behavior?
* Is there a relationship between **total bill and tip**?
* Do tips vary depending on the **day of the week**?

---

# 🎯 Objectives

✔ Understand the structure of the dataset
✔ Clean and prepare the data for analysis
✔ Explore individual variable distributions
✔ Identify relationships between variables
✔ Generate insights that can help restaurants improve service strategies

# 📊 EDA Workflow (7-Step Framework)

### 1️⃣ Understand the Problem

Define business questions and analysis objectives.

### 2️⃣ Load & Inspect Data

* Load dataset
* Inspect shape, columns, and data types
* Preview rows

### 3️⃣ Data Cleaning

* Check missing values
* Remove duplicates
* Validate data types

### 4️⃣ Univariate Analysis

Analyze individual variables using:

* Histograms
* Box plots
* Summary statistics

### 5️⃣ Bivariate Analysis

Explore relationships between variables:

* Scatter plots
* Group comparisons

### 6️⃣ Multivariate Analysis

Understand multiple relationships using:

* Correlation heatmaps
* Pair plots

### 7️⃣ Insights & Storytelling

Summarize patterns and provide recommendations.

---

# 📈 Key Insights

Some observations from the analysis include:

* Tip amount generally increases with **total bill amount**
* **Dinner customers** tend to tip more than lunch customers
* Larger **party sizes** often result in higher total tips
* Tipping behavior may vary depending on **day of the week**
